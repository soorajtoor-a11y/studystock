# HOSA Study Outlines — Official 2025–2026 (ALL WAVES)

Every HOSA competitive event that has a written test, rebuilt from the official HOSA
Competitive Event Guidelines 2025-2026. **46 event outlines** in total.

Each `event-outline.txt` uses the same `A. <Area> (<N> items — <%>)` format as the FBLA/DECA
outlines, so it parses and feeds `generate_bank.py` the same way.

## What's official vs. representative
- **Official (verbatim from the HOSA guideline):** test format, the content-area TEST PLAN
  with percentages, and the reference textbooks. Item counts = the official % applied to the
  50-question test (Hamilton rounding; each sums to 50).
- **Representative:** the per-area objectives (aligned to each official area + references).

## Events WITHOUT a percentage plan (5)
These have a written test but HOSA publishes no % breakdown — outlines list official areas +
references, no item counts: Parliamentary Procedure, Biomedical Debate, Medical Reading,
Healthcare Issues Exam, HOSA History Exam. (Biomedical Debate and Medical Reading change topic
each year — verify the current reading list/topic.)

## Events NOT included (no written test → belong elsewhere)
- **Personal Care, Life Support Skills** — skills-only demonstration events (no written test).
- **Clinical Specialty** — portfolio + presentation event (no written test).
- **Public Health** — presentation event (belongs in the presentation grader).

## Folders to DELETE (not real HOSA events)
- **Exercise Science, Psychology, Sociology** — not HOSA competitive events (confirmed).
- **Cell Biology & Histology** — not a standalone HOSA ATC test (folded into the Biology ATC).

## Folders to CONFIRM (likely duplicates)
- `medical-terminology-academic`, `nutrition-academic`, `pharmacology-academic` — appear to
  duplicate events already covered (Medical Terminology, Nutrition, Pharmacology). Confirm and
  remove if redundant.

## Note on Organizational Leadership
Two same-named events exist: the NATIONAL ATC "Leadership" test (Covey/Maxwell, 50/50 — used
here) and a Texas voting-delegate variant with different content. Confirm which your program uses.

Source: official HOSA 2025-2026 guidelines via state/Canada HOSA mirrors (Texas, SC, Florida,
HOSA Canada) + the NGL/Cengage ATC guideline — hosa.org itself blocks automated fetching.
